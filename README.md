## 🚀 Tecnologias

Esse projeto foi construído com:

- *HTML5*  
- *CSS3*  
- *JavaScript*

## 📌 Sobre o Projeto

Aplicação web que permite **agendar serviços de uma manicure** de forma prática e rápida. O usuário pode selecionar datas e horários com facilidade, trazendo uma experiência semelhante ao que se encontra em sistemas reais de agendamento online.  

👉 [Acesse o projeto aqui](https://jonatas-b.github.io/Agendamento-Manicure/)
